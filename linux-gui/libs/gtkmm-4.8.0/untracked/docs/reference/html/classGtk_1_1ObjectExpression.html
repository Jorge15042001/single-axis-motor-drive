template <class T >
static Glib::RefPtr<ObjectExpression<T> > Gtk::ObjectExpression< T >::create ( const T &  object)
static

Creates an expression evaluating to the given object with a weak reference.

Once the object is disposed, it will fail to evaluate. This expression is meant to break reference cycles.

If you want to keep a reference to object, use Gtk::ConstantExpression::create().

Parameters
objectObject to watch.
Returns
A new Gtk::Expression.
