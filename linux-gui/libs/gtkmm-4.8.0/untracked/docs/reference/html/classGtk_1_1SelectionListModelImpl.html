Detailed Description

Gtk::SelectionListModelImpl is an object that implements the Gio::ListModel and Gtk::SelectionModel interfaces.

gtk_stack_get_pages() returns an object of a class that implements both GListModel and GtkSelectionModel. Since that C class is not public, it's not wrapped in a C++ class. A C object of such a class can be wrapped in a Gtk::SelectionListModelImpl object. Gtk::SelectionListModelImpl does not directly correspond to any GTK class.

This class is intended only for wrapping C objects returned from GTK functions.

See also
Gtk::Stack::get_pages()
